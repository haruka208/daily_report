## ・わかったこと
- プリミティブ型　文字列型、数値型、論理型、未定義　値そのものが変数に入っている、const で定義したものは値を変更することができない

- オブジェクト型　オブジェクト、配列、関数などなど・・プリミティブ型以外　実体はメモリのどこかに置き、その“参照（住所）”が変数に入る、const で定義しても key や要素を変更することができる

- アロー関数
const outputProductInfo = (productName, productPrice) => {
  console.log("こんにちは");
  console.log(`${productName}の値段は、${productPrice}円です`);
};

-  三項演算子
let userAge = 25;
let userType = userAge > 20 ? "adult" : "child"
console.log(userType); // adult

- スプレッド構文:全く別のオブジェクトが生成される
const product1 = {
  name: "りんご",
  price: 100,
};
const product3 = { ...product1 }

- 配列もオブジェクトと同様にスプレット構文を使用できる
const cloneProducts = [...products];
console.log(cloneProducts);
